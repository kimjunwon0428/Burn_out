# Burnout

직장인의 번아웃을 테마로 한 **2D 액션 로그라이트 + 소울라이크** 게임

## 📖 프로젝트 소개

**Burnout**은 거대한 오피스 타워가 '번아웃' 에너지로 뒤틀려 무한 던전화된 세계를 배경으로, 진실을 알아챈 직장인이 퇴사를 위해 꼭대기의 회장에게 돌진하는 이야기를 담은 액션 게임입니다.

- **장르**: 2D 액션 로그라이트 + 소울라이크
- **플랫폼**: PC
- **개발 엔진**: Unity 6000.2.10f1

## ✨ 게임 특징

### 핵심 메커니즘
- **패링·회피 중심의 빠른 템포 전투**: 스태미나 없이 자유롭게 액션을 펼치세요
- **퍼펙트 가드/닷지 시스템**: 정확한 타이밍으로 적의 공격을 막거나 회피하면 강력한 보상 획득
- **로그라이트 시스템**: 매 플레이마다 다른 아이템 조합과 빌드로 새로운 경험
- **턴 기반 전투 구조**: 플레이어 턴과 적 턴이 명확하게 구분되는 전략적 전투

### 성장 시스템

**휘발성 성장 (Run별 초기화)**
- **아이템**: 등급과 중첩으로 직관적 강화, 조합에 따른 시너지
- **이벤트**: 위험·보상 선택으로 일시적 버프·패널티 부여
- **상점**: 일반 상점과 대가를 지불하는 고위험 상점

**영구 성장**
- **허브 업그레이드**: 반복 플레이로 재화 획득 후 단계적 해금
- **아이템 해금**: 플레이 진행에 따라 새로운 아이템 추가

### 플레이스타일

#### 🏃 경량
- 높은 공격 및 이동속도
- 회피로 획득한 자원을 소모하는 강력한 특수기
- 빠른 전투 템포, 높은 순간 화력

#### 🛡️ 중량
- 높은 기초 공격 및 방어력
- 낮은 공격 및 이동속도
- 안정적인 맞딜, 낮은 조작 난이도

#### ⚔️ 가드
- 가드·퍼펙트 가드 기반의 카운터 플레이
- 적의 내구력 상쇄 특화
- 높은 처형 가동률

#### 💰 스캐빈저
- 획득 아이템·이벤트 기대값 극대화
- 성장 사이클에 집중
- 높은 시너지 잠재력

## 🌍 세계관

거대한 오피스 타워가 '번아웃(Burnout)' 에너지로 뒤틀려 무한 던전화되었습니다. 진실을 알아챈 주인공은 무사히 사직서를 제출하고 탈출하기 위해, 동료들의 몬스터화를 멈추기 위해 결단을 내립니다.

**왜곡된 공간들:**
- 회의실 (보고서 지옥)
- 탕비실
- 서버실
- 엘리베이터홀
- 인사평가실
- 회장님의 서재 (보스전)

## 🛠️ 기술 스택

- **게임 엔진**: Unity 6000.2.10f1
- **언어**: C#
- **그래픽**: 2D (팔레트 대비 - 배경 무채색 vs 네온 VFX)


## 🎮 게임 조작 (예정)

- **이동**: WASD
- **점프**: Space
- **약공격**: 마우스 좌클릭
- **강공격**: 마우스 우클릭
- **특수 공격**: Ctrl + 마우스 좌클릭/마우스 우클릭
- **가드**: E (퍼펙트 가드: 타이밍에 맞춰 누르기)
- **닷지**: Shift (퍼펙트 닷지: 타이밍에 맞춰 누르기)

> 구체적인 키 매핑은 개발 진행에 따라 변경될 수 있습니다.

## 🎯 목표 및 차별점

- **직관적 성장**: 아이템 획득 즉시 체감되는 강화
- **쾌감 전투**: 퍼펙트 가드/회피 보상 극대화로 '잘 피하고 잘 막는' 재미 강조
- **낮은 진입 장벽**: 불합리 요소 제거, 경쾌한 템포로 도전 유도
- **로그라이트 요소**: 제한적 영구 성장으로 점진적 난이도 극복
